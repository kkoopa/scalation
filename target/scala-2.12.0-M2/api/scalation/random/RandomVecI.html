RandomVecI

Related Doc: package random

case class RandomVecI(count: Int = 10, max: Int = 20, skip: Int = 1, unique: Boolean = true, stream: Int = 0) extends VariateVec with Product with Serializable

The RandomVecI class generates a random vector of integers. Ex: (3, 2, 0, 4, 1) has count = 5 and max = 4.

count

the size of the vector (number of integer elements)

max

generate integers in the range 0 (inclusive) to max (inclusive)

skip

skip this number, i.e, do not use it

unique

whether the integers must be unique

stream

the random number stream
