Related Doc: package analytics

object BayesClassifier

The BayesClassifier object provides factory methods for building Bayesian classifiers. The following types of classifiers are currently supported: NaiveBayes - Naive Bayes classifier SelNaiveBayes - Selective Naive Bayes classifier AugNaiveBayes - Augmented Naive Bayes classifier AugSelNaiveBayes - Augmented Selective Naive Bayes classifier
