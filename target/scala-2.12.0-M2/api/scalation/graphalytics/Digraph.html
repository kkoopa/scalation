class Digraph extends Cloneable

The Digraph class stores vertex-labeled directed graphs using an adjacency set ('ch') representation, e.g., ch = { {1, 2}, {0}, {1} } means that the graph has the following edges { (0, 1), (0, 2), (1, 0), (2, 1) }. Optionally, inverse adjacency via the 'pa' array can be stored at the cost of nearly doubling the storage requirements. ----------------------------------------------------------------------------
